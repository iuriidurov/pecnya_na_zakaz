✅ ШАГ 1: Загрузка файлов

1. Зайдите в панель Beget: https://cp.beget.com
2. Откройте "Файловый менеджер"
3. Перейдите в папку: pesnya-pod-zakaz.ru/public_html/
4. Создайте БЭКАП старой версии (если есть):
   - Выделите все файлы
   - Нажмите "Архивировать"
   - Скачайте архив
5. Удалите старые файлы из public_html/
6. Загрузите ВСЕ файлы из этой папки:
   - index.html
   - policy.html
   - robots.txt
   - sitemap.xml
   - yandex_verification.html
   - .gitignore
   - Папки: api/, css/, js/, img/, audio/

⚠️ УБЕДИТЕСЬ, ЧТО ЗАГРУЖЕНЫ ВСЕ ФАЙЛЫ В ПАПКЕ api/:
